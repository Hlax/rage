# RGE Handoff Report

Create or update the latest agent handoff report.

Include:

- Current branch.
- Current phase.
- Current ticket.
- Changed files.
- Implemented behavior.
- Commands run.
- Passing tests.
- Failing tests.
- Safety concerns.
- Spec deviations.
- Recommended next ticket.
- Exact next prompt for the next agent.

Save to:

```txt
agent_reports/<date>_<phase>_<ticket>_handoff.md
```
